# Origami Designer & 3D Fold Animator

Web-based origami designer inspired by Robert Lang's TreeMaker. Create crease patterns and watch paper fold step-by-step in 3D.

## Usage

Open `index.html` in any browser. No server or build step needed.

## Features

- **3-panel layout**: crease pattern editor, 3D viewport, step sequencer
- **Draw mode**: manually draw mountain/valley fold lines with snap-to-grid
- **Tree mode**: Lang's tree method — draw a stick figure, compute crease pattern via circle-river packing
- **6 presets**: Crane, Boat, Frog, Fish, Star, Insect
- **Step-by-step animation**: A-Z labeled fold steps with play/pause/scrub
- **Rigid hinge folding**: crisp paper-like folds (not soft deformation)
- **Real-time validation**: Kawasaki theorem, Maekawa theorem, disk packing checks
- **SVG export**: download crease patterns as SVG files
